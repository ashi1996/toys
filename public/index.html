Documentation - Users collection

Get Users

* Request Type
--> http://localhost:3000/users GET
Description: GET 10 first users collection in the data.
* Request Type
--> /users/?page={number} GET
Description: Navigation between collection pages.

User structure:

role": "Regular/Admin",
"date_created":"2021-02-10T20:42:24.885Z",
"name": "User",
"email": "User@gmail.com",
"password":****

Add Users

* Request Type
--> http://localhost:3000/users POST
Description: POST request to add a new user.

User Schema to add user:

role": "Regular/Admin", <--------> "permission"
"name": "User", <--------> "required"
"email": "User@gmail.com",<-------->"required"
"password":**** <-------->"required"

Users Login

* Request Type
--> http://localhost:3000/users/login POST
Description: login request to create a new token.

User Schema to do login:

"email": "User@gmail.com",<-------->"required"
"password":**** <-------->"required"

Documentation - Toys collection

Get Toys

* Request Type
--> http://localhost:3000/toys GET
Description: GET 10 first Toys collection from the data.
* Request Type
--> /toys/?page={number} GET
Description: Navigation between collection pages.
* Request Type
--> /toys/?s={String} GET
Description: Search letter combination at name/category.
* Request Type
--> /toys/price/?min={Number}&?max={Number} GET
Description: Search at price document who is between min and max.

Toys structure:

"info": "Toy information",
"date_created":"2021-02-10T20:42:24.885Z",
"name": "Toy Name",
"category": "Toy Category",
"img_url": URL
"price": Number
"user_id":Id of product owner

Add Toys

* Request Type
--> http://localhost:3000/toys POST
Description: POST request to add a new toy.
p.s. Additionally token must be sent!

Toy Schema to add a new toy:

"info": "Toy information", <-------->"required"
"name": "Toy Name", <-------->"required"
"category": "Toy Category", <-------->"required"
"img_url": URL <-------->"required"
"price": Number <-------->"required"
"user_id":Id of product owner <-------->"required"

Put Toys

* Request Type
--> http://localhost:3000/toys/{Id Toy} PUT
Description: PUT request to edit a toy.
p.s. Additionally token must be sent!

Toy Schema to edit a toy:

"info": "Toy information", <-------->"required"
"name": "Toy Name", <-------->"required"
"category": "Toy Category", <-------->"required"
"img_url": URL <-------->"required"
"price": Number <-------->"required"
"user_id":Id of product owner <-------->"required"

Del Toys

* Request Type
--> http://localhost:3000/toys/{Id Toy} DELETE
Description: DELET request to delet a toy. -
p.s. Additionally token must be sent! -
