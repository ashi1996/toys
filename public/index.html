Documentation - Users collection

Get Users

* Request Type
--> https://toys-ashi.herokuapp.com/users GET
Description: GET 10 first users collection in the data.
* Request Type
--> /users/?page={number} GET
Description: Navigation between collection pages.

User structure:

role": "Regular/Admin",
"date_created":"2021-02-10T20:42:24.885Z",
"name": "User",
"email": "User@gmail.com",
"password":****

Add Users

* Request Type
--> https://toys-ashi.herokuapp.com/users POST
Description: POST request to add a new user.

User Schema to add user:

role": "Regular/Admin", <--------> "permission"
"name": "User", <--------> "required"
"email": "User@gmail.com",<-------->"required"
"password":**** <-------->"required"

Documentation - Toys collection

Get Toys

* Request Type
--> https://toys-ashi.herokuapp.com/toys GET
Description: GET 10 first Toys collection from the data.
* Request Type
--> /toys/?page={number} GET
Description: Navigation between collection pages.
* Request Type
--> /toys/?s={String} GET
Description: Search letter combination at name/info.
* Request Type
--> /toys/cat/ {Toys Category} GET
Description: Search a toy, by his category.
Cat Pagination /toys/cat/ {Toys Category}/?page={Number}
Description: Search a toy, by his category, and Pagination together.
* Request Type
--> /toys/prices/?min={Number}&?max={Number} GET
Description: Search at price document who is between min and max.
Filter Pagination /toys/prices/?min={Number}&?max={Number}&page={Number}
Description: Filter price and Pagination together.

Toys structure:

"info": "Toy information",
"date_created":"2021-02-10T20:42:24.885Z",
"name": "Toy Name",
"category": "Toy Category",
"img_url": URL
"price": Number
"user_id":Id of product owner

Add Toys

* Request Type
--> https://toys-ashi.herokuapp.com/toys POST
Description: POST request to add a new toy.
p.s. Additionally token must be sent!

Toy Schema to add a new toy:

"info": "Toy information", <-------->"required"
"name": "Toy Name", <-------->"required"
"category": "Toy Category", <-------->"required"
"img_url": URL <-------->"required"
"price": Number <-------->"required"
"user_id":Id of product owner <-------->"required"

Put Toys

* Request Type
--> https://toys-ashi.herokuapp.com/toys/{Id Toy} PUT
Description: PUT request to edit a toy.
p.s. Additionally token must be sent!

Toy Schema to edit a toy:

"info": "Toy information", <-------->"required"
"name": "Toy Name", <-------->"required"
"category": "Toy Category", <-------->"required"
"img_url": URL <-------->"required"
"price": Number <-------->"required"
"user_id":Id of product owner <-------->"required"
